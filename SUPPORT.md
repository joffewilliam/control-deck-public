# Support

**GitHub Issues is the support channel**, and license or billing questions go to
license@ctrldeck.dev.

- 🐞 Bugs / problems: [open an issue](https://github.com/joffewilliam/control-deck-public/issues/new/choose)
- 💬 Feedback: use the **Feedback** issue template
- 🔑 License, activation, or refund help: license@ctrldeck.dev
- In the app: **Settings → Help & Feedback** (includes a "Copy diagnostics" button
  and a link to your logs folder)

When filing an issue, please include the diagnostics block from
**Settings → Help & Feedback → Copy diagnostics** and describe what you were doing.

**Do not** post API keys, OAuth tokens, license keys, private prompts, customer
data, or full terminal transcripts in public issues.

Control Deck is built by a solo developer, so response times will vary. Thanks for
helping make Control Deck better.
